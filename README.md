# Training Planner

## Автор: Губайдулина Арина 

## Описание

Приложение для планирования и учёта тренировок с графическим интерфейсом. Позволяет добавлять тренировки с указанием даты, типа и длительности, фильтровать записи по этим параметрам и сохранять/загружать данные в формате JSON.

## Технологии

- Python 3.x.
- Tkinter (стандартная библиотека).
- JSON.
- Git.

## Требования

- Python 3.x.
- Библиотека tkinter (входит в стандартную поставку).

## Запуск

1. Клонируйте репозиторий.
2. Перейдите в папку проекта.
3. Запустите: `python main.py`.

## Примеры использования

1. Введите дату (например: 2026-05-02), тип тренировки (например: Кардио), длительность (например: 45).
2. Нажмите «Добавить тренировку».
3. Используйте фильтры для поиска по типу или дате.
4. Данные сохраняются в `data/trainings.json`.
